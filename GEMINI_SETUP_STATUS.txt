QUICK START (5 MINUTES)
================================================================================

1. GET API KEY
   Visit: https://aistudio.google.com/app/apikey
   Click "Create API Key" and copy the key

2. CONFIGURE ENVIRONMENT
   Create/update .env with:
   GEMINI_API_KEY=your_api_key_here
   GEMINI_MODEL=gemini-1.5-flash

3. INSTALL & START
   pip install -r requirements.txt
   python app.py

4. TEST
   Open: http://localhost:5000
   Click chatbot widget (bottom-right corner)
   Send a test message
   Verify Gemini responds

================================================================================
CONFIGURATION OPTIONS
================================================================================

GEMINI_API_KEY       - Your Google AI Studio API key (REQUIRED)
GEMINI_MODEL         - Model to use (default: gemini-1.5-flash)
CHATBOT_RATE_LIMIT   - Messages per hour (default: 100)
CHATBOT_MAX_HISTORY  - Previous messages for context (default: 20)

Available Models:
  gemini-1.5-flash   - Fast, cost-effective (RECOMMENDED)
  gemini-1.5-pro     - Advanced reasoning, higher cost
  gemini-2.0-flash   - Latest features

================================================================================
TROUBLESHOOTING
================================================================================

Issue: "Gemini API key not configured"
→ Get key from https://aistudio.google.com/app/apikey
→ Add GEMINI_API_KEY to .env
→ Restart app

Issue: "ResourceExhausted" (quota exceeded)
→ Exceeded free tier rate limit (~60 requests/minute)
→ Wait 1 minute or upgrade to paid plan

Issue: App won't start
→ Run: pip install -r requirements.txt
→ Verify Gemini SDK: python -c "import google.generativeai; print('OK')"
